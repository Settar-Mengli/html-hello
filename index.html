Skills

Browse and manage the reusable skill catalog.

In AgentHub, a skill is a reusable capability — a pluggable behavior such as web browsing, document parsing, or calendar access — that can be attached to an agent at rental time. Skills are versioned independently and priced per use.

Web Search

Browse the web and retrieve up-to-date information from public sources.

Enabled on 3 agents

Document Reader

Parse and summarize PDFs, DOCX, and spreadsheets up to 200 pages.

Enabled on 2 agents

Calendar Manager

Read and create events on connected Google and Outlook calendars.

Enabled on 1 agent

Email Composer

Draft and review emails based on conversational context and tone hints.

Enabled on 2 agents
